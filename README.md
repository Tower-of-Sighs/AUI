- textarea
- html内嵌js
- 简单的svelte
- 浏览器控制台
- 网络通讯和本地通讯
- 多重背景、flex-wrap、text-shadow、渐变、cursor
- checkbox、radio
- animation补全、帧动图
- beta-0.0.1
- 伪元素
- transition 完全适配
- 非线性动画
- 文本可选
- 网络资源
