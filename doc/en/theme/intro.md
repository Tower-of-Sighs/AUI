## Custom Themes

### Creating Your Own Theme

A complete UI theme should usually include at least some of the following:

- Two rectangular regions that share the same color language
- A single-row list item style and a table-cell style
- Multiple button styles with focused and active states
- Multi-line text areas and title text styles
- Input boxes with focused and unfocused states
- Radio buttons, checkboxes, and dropdowns
- Tooltips and dialogs
- Progress bars
- Loading animations

That said, missing some of these is not a big problem. Minecraft UI rarely needs the full set anyway.

Themes can be divided into single-tag themes and composite themes.  
Single-tag themes are mostly pure CSS. Composite themes depend on JS and are currently more cumbersome to build. It makes more sense to revisit that once more front-end framework support has landed.

So this section mainly introduces built-in themes and theme ideas related to ApricityUI.
